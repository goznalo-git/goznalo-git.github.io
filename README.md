# Portfolio

My personal portfolio website.

**Live site:** [gonzalo.co.uk](https://goznalo-git.github.io)


## Tech Stack

- Vanilla HTML, CSS, JavaScript
- Markdown content
- GitHub Pages hosting

## Development

```bash
# Serve locally
python -m http.server 8000 -d docs/
```

## Structure

- `docs/index.html` - Main page
- `docs/*.md` - Content (about, publications, resume)
- `docs/styles.css` - Styling
- `docs/script.js` - Functionality

Based on [jrosser](https://jrosser.co.uk/#about)'s template, who was inspired by [astro-theme-cactus](https://astro-cactus.chriswilliams.dev/) :)
